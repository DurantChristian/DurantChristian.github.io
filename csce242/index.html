CSCE 242: Website Applications

Christian Durant

The website applications class focuses on web technologies to support client-server computing. It also, involves implementation of client-server applications. The students will be utilizing HTML and CSS to design the front-end of a website; matching a given specification. Students will be using javaScript to add functionality to their website. On the back-end, students will learn to utilize node and sever side javaScript to store and retrieve data from a server. Students are able to interact with the instructor, as well as interact with other students, during in class discussions. The students will be engage in course content by completeing programming assignments, projects, and participation activities. The learning outcome of the class will consists of understanding the fundamentals of source control, being able to create an environment for successful website development using HTML, CSS, and javaScript, able to apply object-oriented software development to a javaScript environment, able to gather and parse info contained in JSON, able to create server side applications, and being able to apply the fundamentals of client and server side computing and create applications which exchange data.
